إعداد القوائم المالية

يبدأ إعداد القوائم المالية عمليا بإعداد قائمة الدخل ( الأرباح أو الخسائر ) ثم إعداد قائمة المركز المالي ( الميزانية العمومية ) ثم قائمة التدفقات النقدية ( قائمة المصادر و الإستخدامات ) ،، ثم قائمة التغيرات في حقوق الملكية ( الشركاء أو المساهمين أو المالك الفرد )

مـعـادلـة الميزانية : الأصــول (الموجودات ) = الخصــوم ( الإلتـزامات ) + حقوق الملكية

قائمة الدخل ( قائمة الأرباح أو الخسائر ) :

يتلخص إعداد قائمة الدخل للمنشأة في بيان نتيجة أعمال المنشأة خلال فترة محددة سواء ربح أو خسارة .

إعداد قائمة الدخل يتطلب من المحاسب إعداد جدول الأصول الثابتة حتى يتمكم من حساب الإهلاك ، و حساب صافي الأصول الثابتة ،،

  • صافي الأصول ـــــــــــــــ إلى المركز المالي
  • الإهلاك ــــــــــــــــ قائمة الدخل
  • صافي الإيرادات ( صافي المبيعات )
  • إيرادات النشاط الرئيسي للمنشأة بالصافي
  • المبيعات مطروحا منها مردودات المبيعات
  • تكلفة المبيعات ( تكلفة الإيرادات )
  • قيمة المواد الخام أو المشتريات مضافا إليها كل التكاليف التي تكلفتها البضاعة حتى تكون جاهزة للبيع :

سواء مواد خام للتصنيع أو جمارك على المشتريات أو النقل و الشحن و التغليف و التالف المبيعات – تكلفة المبيعات = مجمل الربح

مجمل الربح = المبيعات ( الإيرادات ) – تكلفة المبيعات ( تكلفة الإيرادات )

مجمل الربح مطروحا منه:

مصروفات التشغيل ، مصروفات البيع و التسويق ، إستهلاكات الأصول الثابتة ، المصروفات الإدارية و العمومية = صافي الربح قبل الزكاة أو الضريبة

يضاف إليه: إيرادات أخرى ( مختلفة عن إيرادات النشاط )

يخصم منه: مصروفات أخرى ( غير مصروفات التشغيل و التسويق و الإهلاك و الإدارية ) = صافي الربح المعدّل

يخصم منه: الزكاة أو الضرائب = صافي الربح

يتم إضافة صافي الرباح أو الخسارة إلى الأرباح أو الخسائر المرحّلة سابقا ،، و يتم إثبات الأصول الثابتة في قائمة المركز المالي.

قائمة المركز المالي ( الميزانية العمومية ) :

إعداد قائمة المركز المالي ( الميزانية العمومية ) يتطلب من المحاسب أو الإدارة المالية أو فريق العمل المختص بإعداد الميزانية القيام بما يلي :

خطوات اعداد الميزانية العمومية :

  1. الإنتهاء من جميع القيود المحاسبية الخاصة بالفترة التي سيتم إعداد الميزانية العمومية لها.
  2. إقفال وتصفير جميع الحسابات الوسيطة.
  3. إثبات المصروفات المستحقة ومراجعتها بدقة.
  4. إثبات المصروفات المدفوعة مقدما ومراجعتها بدقة.
  5. الإنتهاء من حساب المرتبات و الأجور وإثبات المرتبات والأجور المستحقة.
  6. إثبات إهلاك الأصول الثابتة كمصروف ،، و إعداد جدول الأصول الثابتة كما في إيضاح الأصول الثابتة.
  7. عمل ميزان مراجعة تفصيلي بعد كل التسويات و القيود السابقة.
  8. إعداد قائمة الدخل ،،،،،،،،،،،،،، الأرباح و الخسائر.
  9. إعداد قائمة المركز المالي ،،،،،،،،،،، الميزانية العمومية.
  10. إعداد قائمة التغير في حقوق الملكية.
  11. إعداد قائمة التدفقات النقدية ،،،،،،،،، قائمة المصادر و الإستخدامات.

أسلوب عرض قائمة المركز المالي أو الميزانية يسهّل على قاريء القوائم المالية فهمها:

  • حسابات الأصول ( متداولة وثابتة ) لابد أين يكون لها إيضاحات شارحة و موضحة أسباب نشأة و تكوين و زيادة أو نقصان أرصدة هذه الحسابات ما عدا حسابات العملاء حتى لا تنكشف أسماء أو حركات العملاء للمنافسين.
  • يتم اظهار المخزون بالتكلفة أو سعر السوق أيهما أقل أو متوسط التكلفة بدون تفصيل حفاظا على أسرار المنشأة.
  • ويتم إظهار الحسابات الدائنة بدون تفاصيل تكشف أسرار المنشأة و خاصة حسابات الموردين.

قائمة التدفقات النقدية :

قائمة التدفقات النقدية توضح: مصادر و استخدامات الأموال.

مصادر الأموال من : الأنشطة التشغيلية ، و الأنشطة الإستثمارية ، و الأنشطة التمويلية.

استخدامات الأموال في : تمويل الأصول الثابتة ، الأصول المتداولة ، سداد الإلتزامات.

قائمة التدفقات النقدية هي أكثر القوائم المالية خطورة في فهم حالة المنشأة الحالية و المستقبلية لها ، حيث أن العجز النقدي في أغلب الأحيان يؤدي إلى عجز مالي و من ثم إلى خسائر.
